✅ ToDo List App
A clean and minimal To-Do List web application built with HTML, CSS, and JavaScript. It helps you manage your daily tasks with a simple and elegant interface.

🌐 Live Demo
👉 amanyhisham.github.io/ToDo-APP

✨ Features

➕ Add new tasks easily
✔️ Mark tasks as completed
🔍 Search/filter tasks by name
📂 Filter view: All Tasks, Active, Completed
🗑️ Clear all completed tasks with one click
📊 Track progress (e.g., "2 of 5 completed — 40%")
🌙 Dark mode toggle
💾 Tasks saved in localStorage (persist after refresh)
