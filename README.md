 # ✅ ToDo List App

A clean and minimal **To-Do List** web application that helps you manage your daily tasks, built with **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Live Demo

🔗 [View Live](https://amanyhisham.github.io/ToDo-APP/)

---

## ✨ Features

- ➕ Add new tasks
- ✔️ Mark tasks as completed
- 🔍 Search tasks by name
- 📂 Filter view: **All Tasks**, **Active**, **Completed**
- 🗑️ Clear all completed tasks with one click
- 📊 Track progress (e.g., "2 of 5 completed — 40%")
- 🌙 Dark mode UI
- 💾 Data saved in localStorage

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amanyhisham/ToDo-APP.git
   ```

2. **Open the project:**
   ```bash
   cd ToDo-APP
   open index.html
   ```

   > No installation or build step needed — runs directly in the browser!
